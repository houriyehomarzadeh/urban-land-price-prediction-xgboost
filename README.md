# Urban Land Price Prediction Using XGBoost

## Overview

This project predicts urban land prices using socioeconomic, demographic, housing, and land-use indicators. The model is developed using XGBoost Regression and evaluated using multiple performance metrics.

## Features

* Data preprocessing with Scikit-learn Pipelines
* Missing value imputation
* Feature scaling and categorical encoding
* XGBoost regression model
* Cross-validation
* Model performance evaluation

## Evaluation Metrics

* RMSE (Root Mean Squared Error)
* MAE (Mean Absolute Error)
* R˛ Score
* Pearson Correlation
* 5-Fold Cross Validation

## Technologies

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* Matplotlib
* SciPy

## Data

The dataset used in this project is proprietary and cannot be publicly shared.

To run the project, users should provide a dataset with the same structure and variable names used in the code.
